# 📊 Analytics Dashboard (Intern Project)

A modern and responsive Analytics Dashboard built with **React.js**, **Tailwind CSS**, and **Recharts**. The dashboard provides interactive charts, dynamic filtering, searching, sorting, and pagination for analytics data.

## 📊 Dashboard Modules

### Dashboard Cards

- Total Users
- Approved Users
- Failed Users
- Revenue

Cards automatically update according to applied filters.

---

### Charts

# Bar Chart

Shows the number of users according to status.

# Pie Chart

Displays status distribution visually.

---

### Data Table

Includes

- Search
- Sorting
- Pagination
- Status
- Date

---

## 🔍 Filters

Users can filter data using

- Status
- From Date
- To Date

All charts and cards update automatically after filtering.

---

## 📄 Pagination

Features include

- Previous Page
- Next Page
- Page Number
- Showing current records

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/tanveerahmed9413/Analytics-Dashboard.git
```

Go to project folder

```bash
cd Analytics-Dashboard
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

---

## 📦 Build

```bash
npm run build
```
